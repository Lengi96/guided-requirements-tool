# Guided Requirements Tool - Web App

Next.js 16 Web-Applikation mit Claude API Integration.

> Siehe [Haupt-README](../README.md) fur die vollstandige Projektdokumentation.

## Setup

```bash
npm install
cp .env.local.example .env.local
# ANTHROPIC_API_KEY eintragen
npm run dev
```

## Scripts

| Script | Beschreibung |
|---|---|
| `npm run dev` | Entwicklungsserver (http://localhost:3000) |
| `npm run build` | Production Build |
| `npm start` | Production Server |
| `npm run lint` | ESLint |
